How to install this... короче братухи вам нужна джава выше java 11 а скачать тут https://adoptium.net/temurin/releases нужно еще скачать мавен и скачать тут https://maven.apache.org/download.cgi
как только скачали maven.zip любой из тех вам нужно его распаковать в любую папку. 
Затем нажать Win + R и ввести sysdm.cpl -> Дополнительно -> Переменные Среды.
В системных переменных найдите Path, нажмите "Изменить" 
Добавь путь к папке bin внутри Maven например C:\Program Files\apache-maven-3.9.6\bin
нажмите ок во всех окнах.
